# 📊 Sistema de Auditoría y Control de Inventarios

FMC es una herramienta robusta diseñada para optimizar y automatizar el control de existencias en entornos comerciales. Su objetivo principal es mitigar los desafíos críticos en la gestión de almacén: **faltantes, sobrantes, productos próximos a vencer y mermas.**

## 🎯 Solución de Problemas (Core)

A diferencia de un registro simple, FMC está orientado a la **auditoría**:
* **Control de Mermas:** Identificación precisa de pérdidas operativas.
* **Gestión de Caducidad:** Alerta y monitoreo de productos con fecha crítica.
* **Trazabilidad:** Registro de movimientos por usuario para auditorías internas.
* **Seguridad:** Validación de acceso y niveles de permisos.

## 🚀 Funcionalidades Clave

* **Monitoreo en Tiempo Real:** Visualización del estado actual de los conteos físicos vs. sistema.
* **Módulo de Usuarios:** Registro detallado de cada acción realizada (quién, cuándo y qué se modificó).
* **Interfaz de Alta Eficiencia:** Desarrollado con una arquitectura ligera (HTML/CSS/JS) para carga rápida en estaciones de trabajo POS.
* **Reportes de Inventario:** Generación de datos listos para análisis y toma de decisiones.

## 🛠️ Stack Tecnológico

* **Backend:** PHP (Lógica de servidor y manejo de sesiones).
* **Frontend:** JavaScript moderno, HTML5, CSS3.
* **Base de Datos:** MySQL con arquitectura de tablas relacionales.
* **Seguridad:** Filtros de saneamiento de datos y control de acceso.

## 📁 Estructura del Proyecto

* `/index.php`: Panel principal y login.
* `/js`: Lógica de interacción y validaciones del lado del cliente.
* `/css`: Estilos personalizados para entorno oscuro/empresarial.
* `/php`: Scripts de procesamiento de datos y consultas SQL.
