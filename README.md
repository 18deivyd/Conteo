# Conteo
Herramienta de gestión de inventario, desarrollada con PHP, HTML, CSS y SQL, la he creado para optimizar y automatizar el control de existencias. Su principal objetivo es proporcionar una solución integral y segura que aborde los desafíos más comunes en la gestión de inventarios, como lo son los faltantes, sobrantes, próximos a vencer y merma. Permite una gestión de seguridad a través de la validación de usuario que será monitoreada a través de su registro
